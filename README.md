# Play
사용자를 향해 전방에서 날아오는 고양이, 아이템들을 시선을 통해 없애는 게임.<br>
화면 중앙의 포인터를 오브젝트에 가져다대면 일정 시간 이후 사라지고, 점수를 얻는다.<br>
나쁜 아이템을 먹을 경우 생명 감소!<br>


# Game Mode
- Day mode <br>
  낮의 도시에서 고양이를 구하라! 
- Night mode
- Score mode
